## 打开chrome的开发者工具

网页右键 -> 最下面的'检查'，就能调出开发者工具

- 直接按`Option + Command + J`（Mac）或者`Ctrl + Shift + J`（Windows / Linux）。
- Chrome菜单-> View -> Developer - > Developer Tools, 或者`Option + Command + I`（Mac）以及`Ctrl + Shift + I`（Windows / Linux）。

### 1. 直接使用标签的ID名字即可调出该标签的dom

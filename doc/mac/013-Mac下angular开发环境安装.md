## 1. install node

```  bash
# 需要事先安装好 brew
~ brew install node
~ npm -v
~ node -v
```
![](./img/037-mac.png)  

![](./img/038-mac.png)   

```    bash
# 使用brew安装的node版本太高了，告知angular2 里面`.sass`文件`node-sass`包不兼容
# 卸载node最新版本
brew uninstall node 
# node官网： https://nodejs.org/en/download/    打开网站选择 LTS 版本的。下载mac版本的，然后店家安装
```
